## MySQL One for all

![MySQL](https://www.tshirtgeek.com.br/wp-content/uploads/2021/08/com031.jpg)

### Proposta Apresentada: 

Mais um projeto de MySQL feito, nesse projeto foi criado um banco de dados, dentro de um container Docker, do inicio ao fim, simulando um banco do Spotify, utilizando tudo que foi aprendido sobre MySql, para criar, editar e consultar um Banco de dados.

</br>

### Conhecimento Utilizado: 

- MySQL </br>
- Docker </br>
- Javascript </br>
